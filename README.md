# OpenChain-ChinaWG

This OpenChain China Workgroup is primarily focused on the region of China. The purpose of this Workgroup is to create a space for gathering, learning and sharing open source subject.

这个 OpenChain China 工作组主要聚焦于中国地区。该工作组的目的是创建一个用于聚集、学习和分享开源主题的空间.
